# Northern Thailand PM2.5 Forecast Web App

เว็บแอปสำหรับแสดงผลการพยากรณ์ค่าฝุ่น PM2.5 รายจังหวัดในภาคเหนือของประเทศไทย ผ่านแผนที่แบบโต้ตอบ พร้อมสีบอกระดับความเสี่ยงและข้อมูลรายชั่วโมง

## Project Overview

โปรเจกต์นี้พัฒนาเป็นเว็บ dashboard สำหรับดูค่าพยากรณ์ PM2.5 ในระดับจังหวัด โดยเน้นการแสดงผลให้เข้าใจง่ายผ่านแผนที่จังหวัด สีระดับความเสี่ยง ตารางอันดับค่าฝุ่น และกราฟแนวโน้ม 24 ชั่วโมง

เว็บนี้เป็น static web application จึงสามารถเปิดผ่าน GitHub Pages ได้โดยไม่ต้องใช้ backend, database, API key หรือขั้นตอน build เพิ่มเติม

## Main Features

- แสดงแผนที่ภาคเหนือพร้อมเส้นแบ่งเขตจังหวัด
- แสดงสีแต่ละจังหวัดตามระดับ PM2.5
- เลือกวันพยากรณ์ได้ตั้งแต่ 15 พฤษภาคม 2569 ถึง 15 พฤษภาคม 2570
- เลือกดูข้อมูลรายชั่วโมงได้ครบ 24 ชั่วโมงของแต่ละวัน
- แสดง tooltip เมื่อชี้ไปที่จังหวัด พร้อมค่า PM2.5, AQI และระดับความเสี่ยง
- แสดงจังหวัดที่มีค่าฝุ่นสูงสุด และค่าเฉลี่ยของภาคเหนือในช่วงเวลาที่เลือก
- แสดงตารางจัดอันดับค่าฝุ่นของแต่ละจังหวัด
- แสดงกราฟแนวโน้ม PM2.5 รายชั่วโมงของจังหวัดที่เลือก
- รองรับการเปิดใช้งานบน desktop, tablet และ mobile

## PM2.5 Risk Levels

| PM2.5 (µg/m³) | ระดับคุณภาพอากาศ | สีที่ใช้บนแผนที่ |
|---:|---|---|
| 0 - 15 | ดีมาก | ฟ้า |
| 15.1 - 25 | ดี | เขียว |
| 25.1 - 37.5 | ปานกลาง | เหลือง |
| 37.6 - 75 | เริ่มมีผลกระทบต่อสุขภาพ | ส้ม |
| มากกว่า 75 | มีผลกระทบต่อสุขภาพมาก | แดง |

## Data Structure

ข้อมูลพยากรณ์ถูกแบ่งเป็นไฟล์รายเดือนเพื่อลดขนาดไฟล์ที่โหลดในครั้งแรก และช่วยให้ใช้งานบนมือถือได้เร็วขึ้น

```text
index.html
styles.css
app.js
.nojekyll
data/
  forecast_index.json
  northern-thailand.geojson
  forecasts/
    2026-05.json
    2026-06.json
    ...
    2027-05.json
```

## Important Note

ข้อมูลพยากรณ์ในโปรเจกต์นี้เป็นข้อมูลจากแบบจำลอง baseline สำหรับการสาธิตและการศึกษา โดยใช้แนวคิดจากฤดูกาล สภาพอากาศ และปัจจัยภูมิประเทศรายจังหวัด ไม่ใช่ข้อมูลตรวจวัดแบบ real-time จากสถานีจริง

หากต้องการนำไปใช้จริง ควรเชื่อมต่อข้อมูล historical PM2.5 และข้อมูลสภาพอากาศจากแหล่งข้อมูลทางการ เช่น Air4Thai, กรมควบคุมมลพิษ หรือบริการ API ด้านคุณภาพอากาศ แล้วนำข้อมูลมาฝึกโมเดลพยากรณ์เพิ่มเติม

## Deployment

โปรเจกต์นี้สามารถ deploy บน GitHub Pages ได้โดยตรง:

1. ไปที่ `Settings`
2. เลือก `Pages`
3. เลือก `Deploy from a branch`
4. เลือก branch `main`
5. เลือก folder `/root`
6. กด `Save`

หลัง deploy แล้ว เว็บจะเปิดได้ผ่าน GitHub Pages URL ของ repository นี้
